# Catatan Bash

Catatan ini membahas tentang segala sesuatu berkaitan dengan codingan Bash Shell.

Source Code untuk menampilkan tanggal dan jam:

```bash
#!/bin/sh
printf "\n"
tanggal=$(date)
printf "$tanggal \n"
```
